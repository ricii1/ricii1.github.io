Student Performance Dataset

  • Open Source Code

Student Performance Dataset is a project that I created to analyze student performance data. This project uses Python with the Pandas and Matplotlib libraries. The dataset used in this project is the Student Performance Dataset from Kaggle. The goal of this project is to analyze the factors that affect student performance and visualize the data to make it easier to understand. I also created a model student's GPA based on the dataset.
